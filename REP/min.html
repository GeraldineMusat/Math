Témprature Minimum en Fahrenheit

Californie

Description

On peut observer dans l'Etat de la Californie que les température minimum au fil des années est assez constant, les température minimale sont pour la pluspart entre 0 et 20 F°.

Montana

Description

Dans l'Etat du Montana nous observons des température minimum irrégulière. Toute les température sont très basse, en dessous de 0 F°.

Kansas

Description

Pour l'Etat du Kansas, nous n'avons pas de données pour étudier la température minimum avant 2008. A partir de cet année nous pouvons observer des température minimum assez régulière avec une variation de température de 8 F° maximum.

New York

Description

Dans l'Etat de New York nous observons un courbe très irrégulière. Les température minimale peuvent varier de plusieurs dizaine de F°.

Floride

Description

Pour l'Etat de la Foride nous avons des piques de température minimale vers le haut. Les température sont assez régulière pendant les années.
